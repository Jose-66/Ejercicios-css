Propiedades CSS utilizadas en las páginas

Colores de fondo

Se usó la propiedad background-color para definir colores de fondo en las distintas secciones y cajas de las páginas.

Bordes y contornos

Para destacar ciertos elementos, se empleó border y border-radius para esquinas redondeadas.

Organización en columnas

Se usaron display: flex o float para colocar elementos en dos columnas.

Tipografía

La tipografía se definió con font-family y se aplicaron tamaños específicos con font-size.
