Explicación de propiedades que se han usado

1. Diseño con Grid y Flexbox

Se ha utilizado display: grid y display: flex para organizar los elementos en columnas y filas.

2. Bordes y Estilos de Caja

Las cajas tienen bordes punteados utilizando border: 2px dashed black. También se usa padding y margin para separar los elementos.

3. Tipografía y Estilos de Texto

Para resaltar el texto, se ha usado font-weight: bold y text-transform: uppercase en los títulos.

4. Colores y Fondos

Se han aplicado diferentes colores con background-color y color en los textos y fondos.

5. Sombras y Relieves

Algunas cajas tienen efectos de sombra con box-shadow: 3px 3px 5px rgba(0, 0, 0, 0.3) para darles profundidad.
